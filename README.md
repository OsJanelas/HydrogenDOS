# ❓ What is Hydrogen-DOS?
HydrogenDOS is a simple DOS made in Assembly x86 with a compiler made in Batch

# 🤔 Fun fact
This OS have a comand called plasma, this comand runs a rgb aurora

# ⚙️ Comands
## HELP
this show all comands, except help

## CLS
this comand clear screen

## REBOOT
reboot system

## PLASMA
rgb aurora animation
